# Virtual Herbal Garden (VR)

An immersive Virtual Reality educational platform designed to showcase medicinal plants and their benefits. This project is built specifically for the AYUSH sector to help users identify and learn about herbal species in a high-fidelity 3D environment.

## 🌿 Project Overview
The Virtual Herbal Garden allows users to explore a digital landscape of medicinal plants. It uses VR to provide an interactive learning experience that is more engaging than traditional textbooks.

- Objective: To bridge the gap between traditional herbal knowledge and modern technology.
- Target Platform: VR (Tested on Meta Quest/PCVR)
- Institution: Poornima Institute of Engineering and Technology (PIET)

## 🛠 Technologies Used
* Engine: Unity 6.2
* Language: C#
* SDK: XR Interaction Toolkit
* 3D Modeling: Blender (Custom plant models)
* AI Tools: Meshy AI, 3D AI Studio (for asset generation)

## 🚀 How to Run the Project
1. Prerequisites: - Install Unity 6.2.x.
   - Ensure you have a VR headset connected (via Link cable or Air Link).
2. Setup:
   - Clone this repository: `git clone https://github.com/YOUR_USERNAME/Virtual-Herbal-Garden.git`
   - Open the project in Unity Hub.
3. Play:    
   - Open the `MainScene` located in `Assets/Scenes/`.
   - Press Play in the Unity Editor or build to your headset via Build Settings.

## 📖 Key Features
* Realistic 3D Models: Detailed plant meshes including the PIET Model and various herbal species.
* Interactive UI: Hover over plants to see their scientific names and medicinal uses.
* Immersive Environment: Optimized for VR performance to ensure a smooth 90 FPS experience.

## 📄 Research & Documentation
This project is part of a broader study on "Augmented Reality-Based AI Assistants for Real-Time Industrial Maintenance and Training" and its application in educational VR environments.
